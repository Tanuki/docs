Usint Tanuki it is also easy to generate embeddings for inputs using embedding models. The Open-AI ada-002 (default) and amazon.titan-embed-text-v1 (see [here](https://github.com/Tanuki/tanuki.py/tree/master/docs/aws_bedrock.md) how to configure to use) models are currently supported out of the box to get embeddings for input data. For embedding output the output typehint needs to be set as  `Embedding[np.ndarray]`. Currently adding align statements to steer embedding model behaviour is not implemented, but is on the roadmap. 

Example with OpenAI Ada model

<CodeBlocks>

  ```bash
    @tanuki.patch

        """
        Scores the input between 0-10
  ```
